# CanChat 💬

A modern full-stack chat application built with a separate frontend and backend architecture, designed for scalable real-time communication and secure user authentication.

# ✨ Features

* 🔐 User Authentication (Signup / Login / Logout)
* 💬 Real-time Chat Interface
* 👤 User Profile Support
* 🌐 REST API Integration
* ⚡ Fast Frontend Rendering
* 📱 Fully Responsive Design
* 🔒 Secure Token-based Authentication
* 🎨 Modern UI Design
